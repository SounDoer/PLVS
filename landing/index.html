Features
Image
System Audio

Native system audio capture

Monitor any audio playing on your machine — no virtual audio cable required. Windows uses WASAPI loopback, macOS uses the native audio tap.

Image
Workspace

Fully flexible layout

Drag any divider to resize panels. Rearrange your workspace with split-tree layout. Pop out individual meters to float on a second screen.

Image
History

Session history & snapshots

Scroll back through the loudness history timeline. Click any moment to freeze all meters at that snapshot — then return to live with one click.

Image
Multichannel

Multichannel support

Works with stereo and surround formats. Automatic channel layout detection with proper per-channel metering and ITU-R BS.1770 loudness for multichannel sources.

macOS — Gatekeeper

PLVS is not notarized by Apple. If macOS blocks the app on first launch, run:

xattr -cr /Applications/PLVS.app
Windows — SmartScreen

If Windows SmartScreen blocks the installer, click More info → Run anyway.
