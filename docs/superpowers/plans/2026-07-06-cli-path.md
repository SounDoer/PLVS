# PLVS CLI PATH Integration Implementation Plan

> **For agentic workers:** implement task-by-task. Keep this plan updated with checkbox progress if work spans multiple turns.

**Goal:** Make installed Windows builds discoverable as `plvs-cli` from a normal terminal.

**Spec:** `docs/superpowers/specs/2026-07-06-cli-path-design.md`

---

## Decisions Locked In

- Add the PLVS install directory to the current user's `PATH`.
- Keep `installMode: "currentUser"`.
- Use Tauri NSIS `installerHooks`; do not copy or replace the full installer template.
- Use PowerShell `[Environment]` APIs from the hook for idempotent add/remove.
- Broadcast an environment change after install and uninstall.
- Verify with the existing Windows installer smoke script.

---
