# Tray Icon Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace PLVS's default app tray icon with a purpose-built letter-P icon (thin stroke, wide rounded bowl, black on transparent) that works as a macOS template image and a Windows dark-mode tray icon.

**Architecture:** A Node.js script renders the SVG P-shape to a 44×44 PNG at `src-tauri/icons/tray.png`. The PNG is registered as a Tauri bundle resource. `useTray.js` loads it via `resolveResource` + `Image.fromPath` at tray creation time and passes it to `TrayIcon.new()` with `iconAsTemplate: true`.

**Tech Stack:** Node.js + `sharp` (PNG generation), `@tauri-apps/api/image`, `@tauri-apps/api/path`, Vitest + @testing-library/react (test)

- [ ] **Step 2: Create the generation script**

Create `scripts/generate-tray-icon.mjs`:

```js
import sharp from "sharp";
import { fileURLToPath } from "url";
import { resolve, dirname } from "path";

const __dir = dirname(fileURLToPath(import.meta.url));
const outPath = resolve(__dir, "../src-tauri/icons/tray.png");

const svg = `<svg xmlns="http://www.w3.org/2000/svg" width="20" height="20" viewBox="0 0 20 20">
  <line x1="5" y1="2" x2="5" y2="18" stroke="black" stroke-width="1.5" stroke-linecap="round"/>
  <path d="M5 2 Q15.5 2 15.5 7 Q15.5 12 5 12" stroke="black" stroke-width="1.5" fill="none" stroke-linecap="round"/>
</svg>`;

await sharp(Buffer.from(svg))
  .resize(44, 44)
  .png()
  .toFile(outPath);

console.log(`Written: ${outPath}`);
```

### Task 2: Register the icon as a Tauri bundle resource

**Files:**
- Modify: `src-tauri/tauri.conf.json`

- [ ] **Step 1: Add `bundle.resources` to tauri.conf.json**

Open `src-tauri/tauri.conf.json`. In the `"bundle"` object, add a `"resources"` field after `"icon"`:

```json
"bundle": {
  "active": true,
  "targets": "all",
  "macOS": {
    "minimumSystemVersion": "14.2"
  },
  "icon": [
    "icons/32x32.png",
    "icons/128x128.png",
    "icons/128x128@2x.png",
    "icons/icon.icns",
    "icons/icon.ico"
  ],
  "resources": [
    "icons/tray.png"
  ]
}
```

Paths in `resources` are relative to `src-tauri/`. `resolveResource("icons/tray.png")` at runtime resolves to this file in both dev and production builds.

```js
(async () => {
  const { running: r, pinned: p, deviceName: d } = creationStateRef.current;
  const menu = await buildMenu({
    running: r,
    pinned: p,
    onToggleCapture: stableToggleCapture,
    onTogglePin: stableTogglePin,
    deviceName: d,
    onToggleWindow: stableToggleWindow,
  });

  const iconPath = await resolveResource("icons/tray.png");
  const icon = await Image.fromPath(iconPath);

  const tray = await TrayIcon.new({
    icon,
    iconAsTemplate: true,
    tooltip: "PLVS",
    menu,
    menuOnLeftClick: false,
    action: (e) => {
      if (e.type === "Click" && e.button === "Left") {
        stableToggleWindow();
      }
    },
  });

  if (!cancelled) trayRef.current = tray;
})();
```
